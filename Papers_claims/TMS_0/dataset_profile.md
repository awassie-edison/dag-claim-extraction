The Tabula Muris Senis dataset is a single-cell transcriptomic atlas of ageing in *Mus musculus* (C57BL/6JN mice), encompassing more than 350,000 cells from 23 tissues and organs across six age groups (1, 3, 18, 21, 24, and 30 months). The dataset was generated using two complementary single-cell RNA sequencing platforms: FACS-based Smart-seq2 (full-length transcripts, sorted into 384-well plates) and microfluidic droplet-based 10x Genomics Chromium (3' end counting). After quality filtering, 110,824 FACS cells and 245,389 droplet cells were annotated. The data are deposited across multiple repositories that collectively represent the same primary dataset: **GEO: GSE132042** hosts the sequencing data; **figshare (doi:10.6084/m9.figshare.8273102.v2)** provides gene count matrices and metadata; **AWS S3 (registry.opendata.aws/tabula-muris-senis/)** stores the raw data files; and an **interactive browser (tabula-muris-senis.ds.czbiohub.org)** allows exploration of the processed data. The 3-month time point overlaps with the previously published Tabula Muris (GEO: GSE109774, Nature 2018) and was used as the reference for semi-automated cell-type annotation. All four repository entries represent different access points to the same underlying primary dataset and are used throughout the paper for every reported analysis.
