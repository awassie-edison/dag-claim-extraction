# Dataset Profile

This paper presents the Tabula Muris Senis (Mouse Ageing Cell Atlas), a single unified single-cell transcriptomic dataset distributed across four repositories. All four entries in the dataset links represent different access points to the same underlying dataset.

**Dataset 1 (main_scrna_geo): GEO GSE132042.** Single-cell RNA sequencing data from the Tabula Muris Senis project, comprising approximately 350,000 cells collected from 23 tissues and organs of male and female C57BL/6JN mice at six age groups (1, 3, 18, 21, 24, and 30 months). The dataset includes two complementary platforms: FACS-sorted cells profiled by Smart-seq2 (110,824 annotated cells) and microfluidic droplet cells profiled by 10x Genomics Chromium (245,389 annotated cells). Tissues covered include bladder, bone marrow, brain (cerebellum, cortex, hippocampus, striatum), fat (brown, gonadal, mesenteric, subcutaneous), heart and aorta, kidney, large intestine, limb muscle and diaphragm, liver, lung, mammary gland, pancreas, skin, spleen, thymus, tongue, and trachea. Sequencing was performed on Illumina NovaSeq 6000. The GEO entry serves as the primary accession for the Gene Expression Omnibus deposit of gene count matrices and metadata.

**Dataset 2 (main_scrna_figshare): figshare (DOI: 10.6084/m9.figshare.8273102.v2).** Processed gene count matrices and cell metadata for the Tabula Muris Senis dataset, providing ready-to-analyse count tables for both the FACS and droplet platforms. This is the same data as GSE132042 but formatted as downloadable processed files for direct computational analysis.

**Dataset 3 (main_scrna_aws): AWS Open Data (https://registry.opendata.aws/tabula-muris-senis/).** Raw data files (FASTQ and related) for the Tabula Muris Senis dataset hosted on a public AWS S3 bucket. This provides access to the unprocessed sequencing reads underlying the GSE132042 dataset, enabling independent reprocessing from raw data.

**Dataset 4 (data_browser): CZ Biohub Data Portal (http://tabula-muris-senis.ds.czbiohub.org/).** An interactive web-based data browser for exploring the entire Tabula Muris Senis dataset, allowing gene expression queries across tissues and age groups. This portal provides a visualization and exploration interface to the same underlying data.

**Relationships between datasets:** All four entries are different access modalities for the identical Tabula Muris Senis dataset. GSE132042 is the canonical accession; figshare provides processed matrices; the AWS bucket hosts raw sequencing files; and the CZ Biohub portal offers interactive exploration. Every analysis in the paper draws from this single coordinated dataset.
