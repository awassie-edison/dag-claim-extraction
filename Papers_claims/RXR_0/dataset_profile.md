This study integrates three datasets generated from APP/PS1ΔE9 mouse brains treated with Bexarotene (RXR agonist) or Vehicle control.

**Dataset 1 (GSE283906): snATAC-seq.** This dataset contains single-nucleus ATAC-seq data from APP/PS1 mouse brains. Ten samples were profiled (n=5 Bexarotene, n=5 Vehicle), comprising both cryopreserved cells from a first cohort and freshly processed cells from a second cohort. Libraries were prepared using the 10x Genomics Chromium Next GEM Single Cell ATAC Kit v2 and sequenced on Illumina NovaSeq S2 PE100. After quality control, 61,353 nuclei were retained. The data were processed with Cell Ranger ATAC, ArchR, Signac, MACS2, and TOBIAS for chromatin accessibility, peak calling, differential accessibility, TF footprinting, and regulatory network construction. This is a primary dataset generated in this study.

**Dataset 2 (GSE289039): RXR ChIP-seq.** This dataset contains chromatin immunoprecipitation sequencing data using an anti-RXR antibody from APP/PS1 mouse brain tissue (n=2 Bexarotene, n=2 Vehicle), with the same treatment paradigm as the single-cell assays. Sequencing was performed on Illumina HiSeq2000. After alignment with Bowtie2 and quality filtering, approximately 35.1 million uniquely mapped reads were obtained for the Bexarotene group and 29.5 million for the Vehicle group. Peak calling was performed with MACS2. This dataset serves as an independent validation layer for RXR binding sites predicted from snATAC-seq footprinting. This is a primary dataset generated in this study.

**Dataset 3 (GSE283905): scRNA-seq.** This dataset contains single-cell RNA-seq data from APP/PS1 mouse brains previously generated by the same group (referenced as Saibro-Girardi et al., bioRxiv 2025). Eight mice were profiled, yielding 37,640 cells after quality control. Libraries were generated using 10x Genomics Chromium and processed with Cell Ranger v7.0 and Seurat v4.2.1. In the current study, these data were re-analyzed for integrative multi-omic analysis: they provided cell-type annotations for snATAC-seq integration, gene expression values for peak-to-gene linkage, chromVAR correlation analyses, and differential expression comparisons that complement chromatin accessibility findings. This is a reanalyzed dataset.

The three datasets are deeply interrelated: snATAC-seq and scRNA-seq were integrated for cell-type annotation, peak-to-gene linkage, and TF motif-expression correlation; ChIP-seq was used to validate RXR binding sites identified from snATAC-seq footprinting; and all three were jointly analyzed to construct the multi-layered RXR regulatory framework.
